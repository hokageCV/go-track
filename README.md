# GoTrack

A command-line task manager built with Go.

### Features

- View all tasks
- Create a new task
- Mark a task as done
- Edit task title
- Delete a task

### Installation

1. Clone the repository
2. Build the project with `go build`
3. Run the executable with `./gotrack`
